미국 거주 한국인 유학생/직장인을 위한 냉장고 기반 집밥 레시피 추천 앱. 이미 가지고 있는 재료와 요리 실력에 맞춰 현실적으로 만들 수 있는 한식 레시피를 추천한다.

## 문제 정의

미국에 사는 한국인들은 집밥을 해먹고 싶어도 한국 식재료가 제한적이고 뭘 만들 수 있는지 모른다. 기존 레시피 앱은 재료를 새로 다 사야 하는 구조인데, BABSIM은 냉장고에 이미 있는 재료를 기준으로 추천한다.

## 대상 사용자

자취하면서 집밥을 해먹고 싶은 미국 거주 한국인 유학생 및 직장인. 요리 초보~중급, H-Mart나 Weee! 같은 한국 마트 이용자.

## 주요 기능

- 냉장고 재료 입력 및 관리
- 보유 재료 + 요리 실력 기반 레시피 추천
- 난이도 필터 (초보 / 중급 / 고급)
- 미국에서 구할 수 있는 재료 기반 한식 레시피 DB

## 기술 스택

- Frontend: Flutter
- Backend: FastAPI (Python)
- Database: PostgreSQL (Supabase)
